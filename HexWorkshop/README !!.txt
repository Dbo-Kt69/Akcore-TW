1. Download Hex Workshop 
http://www.bpsoft.com/downloads/
2. copy DBO-TYPES.hsl and standard-types.hsl into your main catalog of HexWorkshop by default to
C:\Program Files\BreakPoint Software\Hex Workshop v6.8\Structures
3. Edit or create hsl structures by notepad ++
4. Import hsl into structure list in Hex Wokrshop
5. U can easy look for structures and data from Wireshark records (after using dbosniff) or RDFS (after encrypt)
6. Remember that Hex Wokrshop dont support pragma pack(4). U need to manually fill alligment for RDF structures.
